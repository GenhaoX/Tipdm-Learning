提示：如何读取图片和展示图片？

  • （1）读取图片

import matplotlib.pyplot as plt

data = plt.imread("C:/test.jpg") # 需要图片路径C:/test.jpg

  • （2）展示图片 plt.imshow(data, cmap='gray') # 需要图片的数组data， 查看灰度图需要加cmap='gray'参数，查看原图不需要。
